El dataset en cuestin consta de 1000 entradas de diferentes alumnos con las siguientes variables:

  • gender (genero): male / female
  • race/ethnicity (raza/etnia): group A / group B / group C
  • parental level of education (nivel de educacin de los padres): some college / associates degree
  • lunch (descuento del comedor): standard, free / reduced
  • test preparation course prepareacin del test): none / completed
